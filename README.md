## 📌 Projects

### 1. ✓ FocusFlow — To-Do App
> A full-featured task management web application

| | |
|---|---|
| **Folder** | `To-Do-App/` |
| **Live Demo** | [focusflow.onrender.com](https://focusflow.onrender.com) |
| **Stack** | Python, Flask, SQLite, Jinja2, CSS |
| **Status** | ✅ Completed & Deployed |

**Features:**
- Create, edit, delete, and complete tasks
- Priority levels (High / Medium / Low) with color coding
- Categories, due dates, and overdue detection
- Search, filter, and smart task grouping
- Progress bar, task detail pages, responsive design
- Deployed live with Gunicorn on Render
